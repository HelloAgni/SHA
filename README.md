
***How to start***  
```bash
git clone <project>
cd <project>
```

```bash
python -m venv venv
# Windows
. venv/scripts/activate 
# Linux
. venv/bin/activate
python -m pip install --upgrade pip
pip install -r requirements.txt


pip install aiohttp
pip install aiofiles

pip install wemake-python-styleguide
pip install pytest
pip install pytest-asyncio

```
